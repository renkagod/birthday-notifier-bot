# Birthday Notifier Bot 🎂
Telegram-бот для автоматических напоминаний о днях рождения с современным интерфейсом и гибкими настройками.

## Особенности
- **Интерактивный UI**: Управление через Inline-кнопки и пошаговый календарь (Десятилетие -> Год -> Месяц -> День).
- **Умное добавление**: Добавление вручную или выбор контакта из записной книжки.
- **Поддержка Telegram-тегов**: Привязка `@username` для автоматического упоминания в уведомлениях.
- **Гибкие уведомления**:
  - **Плановые**: За месяц, неделю, 3 дня, завтра или сегодня в выбранное вами время.
  - **Оперативные**: За 30 и 5 минут до события.
  - **Полночные**: Поздравление ровно в 00:00 (флаг `-1.0` в настройках).
- **Сортировка списка**: Переключение между алфавитным порядком и порядком наступления праздников.
- **Бэкапы**: Экспорт и импорт вашего списка в формате JSON.
- **Расчет возраста**: Бот автоматически считает, сколько лет исполняется человеку.
- **Docker Ready**: Готов к деплою с настроенными лимитами ресурсов и часовым поясом.

## Быстрый старт (Docker)
1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/renkagod/birthday-notifier-bot.git
   cd birthday-notifier-bot
   ```
2. Настройте конфигурацию:
   ```bash
   cp .env.example .env
   # Укажите ваш BOT_TOKEN в файле .env
   ```
3. Запустите проект:
   ```bash
   docker compose up -d --build
   ```

## Использование
После запуска используйте команду `/start` для вызова главного меню. 
- **📅 Мой список**: Просмотр всех дат. Нажмите "🔄 Сортировать по...", чтобы изменить порядок отображения.
- **⚙️ Настройки**: Установите удобное время для ежедневных напоминаний (например, `10:00`) и выберите нужные интервалы.
- **💾 Бэкап**: Сохраните свои данные в JSON-файл или восстановите их из файла.

## Конфигурация (.env)
- `BOT_TOKEN`: Токен вашего бота от @BotFather.

## Структура проекта
- `bot/main.py` — Логика интерфейса и команд.
- `bot/scheduler.py` — Планировщик уведомлений.
- `bot/database.py` — Хранение данных (SQLite).
- `data/` — Директория с базой данных (создается автоматически).
