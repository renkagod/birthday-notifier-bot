# Birthday Notifier Bot
Telegram-бот для автоматических напоминаний о днях рождения с современным интерфейсом.

## Особенности
- **Интерактивный UI**: Полностью на Inline-кнопках — больше не нужно вводить команды вручную.
- **Умное добавление**:
  - Выбор контакта из записной книжки.
  - Удобный выбор даты через пошаговый календарь (Десятилетие -> Год -> Месяц -> День).
- **Поддержка Telegram-тегов**: Возможность привязать `@username` к имениннику для автоматического упоминания в уведомлениях.
- **Гибкие напоминания**: Уведомления за неделю, 3 дня, 1 день, а также за 30 и 5 минут до события + поздравление в момент наступления.
- **Расчет возраста**: Бот автоматически считает, сколько лет исполняется человеку.
- **Легкий запуск**: Полная готовность к работе в Docker.

## Быстрый старт (Docker)
1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/renkagod/birthday-notifier-bot.git
   cd birthday-notifier-bot
   ```
2. Настройте конфигурацию:
   ```bash
   cp .env.example .env
   # Укажите ваш BOT_TOKEN в файле .env
   ```
3. Запустите проект:
   ```bash
   docker compose up -d --build
   ```

## Использование
После запуска используйте команду `/start` для вызова главного меню. Все управление осуществляется через кнопки:
- **Добавить вручную**: Введите имя и (опционально) тег через пробел (например: `Иван @vanya`), затем выберите дату.
- **Выбрать контакт**: Выберите человека из списка контактов в Telegram.
- **Мой список**: Просмотр всех дат с расчетом возраста и кнопками быстрого удаления.

## Конфигурация (.env)
- `BOT_TOKEN`: Токен вашего бота от @BotFather.

## Структура данных
- `data/` — Директория с базой данных SQLite (создается автоматически).
- База данных теперь поддерживает хранение Telegram-тегов пользователей.
