# Birthday Notifier Bot
Telegram-бот для автоматических напоминаний о днях рождения.

## Особенности
- **Гибкие уведомления**: Напоминания за неделю, 3 дня, 1 день, а также за 30 и 5 минут до события.
- **Простое управление**: Добавление и удаление записей через текстовые команды.
- **Персистентность**: Использование SQLite для надежного хранения данных.
- **Легкий запуск**: Полная готовность к работе в Docker.

## Быстрый старт (Docker)
1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/renkagod/birthday-notifier-bot.git
   cd birthday-notifier-bot
   ```
2. Настройте конфигурацию:
   ```bash
   cp .env.example .env
   # Укажите ваш BOT_TOKEN в файле .env
   ```
3. Запустите проект:
   ```bash
   docker compose up -d --build
   ```

## Конфигурация (.env)
- `BOT_TOKEN`: Токен вашего бота от @BotFather.

## Команды
- `/start` — Инструкция по использованию.
- `/add [Имя] [ДД.ММ.ГГГГ]` — Добавить новую запись (например: `/add Иван 25.05.1990`).
- `/list` — Показать список всех добавленных дней рождения.
- `/delete [Имя]` — Удалить запись по имени.

## Структура данных
- `data/` — Директория с базой данных SQLite (создается автоматически).
